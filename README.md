# FeelBG - Modern Website with Royal Blue & Bronze Gold Design

A stunning, modern website built with the latest HTML5, CSS3, and JavaScript featuring beautiful animations, smooth interactions, and a luxurious royal blue and bronze gold color scheme.

## 🚀 Features

- **Modern HTML5** - Semantic markup with the latest HTML5 standards
- **Advanced CSS3** - Custom properties, gradients, animations, and transitions
- **High-Level JavaScript** - Object-oriented programming with ES6+ features
- **Responsive Design** - Mobile-first approach, works on all devices
- **Smooth Animations** - AOS library integration with custom animations
- **Custom Cursor** - Interactive cursor effects for desktop
- **Dark Mode** - Theme toggle with localStorage persistence
- **Parallax Effects** - Smooth scrolling with parallax backgrounds
- **Performance Optimized** - Lazy loading, debouncing, and throttling
- **Accessibility** - Keyboard navigation and ARIA labels
- **SEO Ready** - Meta tags and semantic HTML structure

## 🎨 Color Scheme

- **Primary (Royal Blue)**: #1e3a8a
- **Secondary (Bronze Gold)**: #b8860b
- **Accent Gold**: #ffd700

## 📦 Installation

1. **Clone or download this repository**

2. **Install dependencies** (optional, for live-server):
```bash
npm install
```

3. **Run the development server**:
```bash
npm start
```
Or simply open `index.html` in your browser.

## ⚙️ Configuration

### Theme Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1e3a8a;
    --secondary-color: #b8860b;
    /* ... more variables */
}
```

### Animation Settings
Adjust AOS settings in `script.js`:
```javascript
AOS.init({
    duration: 1000,
    once: true,
    offset: 100,
    easing: 'ease-out-cubic'
});
```

## 🚀 Performance Tips

1. **Images** - Use optimized images (WebP format recommended)
2. **Fonts** - Self-host fonts for better performance
3. **Lazy Loading** - Images load only when needed
4. **Minification** - Minify CSS and JS for production
5. **CDN** - Use CDN for libraries in production

## 📝 Customization

### Change Colors
Update CSS variables in `:root` selector in `styles.css`

### Add Sections
1. Add HTML section in `index.html`
2. Style in `styles.css`
3. Add interactions in `script.js`
4. Update navigation links

### Modify Animations
- Edit keyframes in `styles.css`
- Adjust AOS attributes in HTML elements
- Modify animation classes in JavaScript
